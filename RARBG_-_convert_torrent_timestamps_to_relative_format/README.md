This script applies to RARBG torrent lists.

It converts torrent timestamps to relative format in local timezone. Also, it recalculates them every 10 seconds.

It uses [moment.js](http://momentjs.com/), [moment-timezone.js](http://momentjs.com/timezone/) and [jsTimezoneDetect](https://bitbucket.org/pellepim/jstimezonedetect)
